UX and UI Design
UX Design refers to the term User Experience Design, while UI Design stands for User Interface Design.

User Interface Design means the way in which user and a website interact in the use of input devices and software. This includes the look and feel, design research, branding, prototyping, adapting to different devices, interactivity, animation.

User Experience design is the process of enhancing customer satisfaction by improving the usability, user friendliness, and an overall pleasing experience provided in the interaction between the customer and the website. It focuses on quality interaction.
For example, the checkout process in an e-commerce website. A UX designer would usually study user's feedback to see how efficient and pleasant the experience is to fill out web forms and make improvements accordingly.

Databases
A place where data from the website is stored and retrieved. Its where all data are collected and organised for rapid search and retrieval by a program. They are structured to facilitate storage, retreival, modification, deletion and other data-processing operations. The popular databases are Oracle, MySQL, MS SQL Server, PostgreSQL, MongoDB, Cassandra.
