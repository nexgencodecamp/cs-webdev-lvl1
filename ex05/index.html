HTML Colour Codes

Color codes are six digits and represent values of red, green and blue. The first two digits are for red, the middle two are for green, and the last two are for blue. These codes are hexadecimal, or base 16.

White RGB code= #FFFFFF 
Blue RGB code = #0000FF  
Red RGB code = #FF0000  
Green RGB code = #00FF00  
Gray RGB code = #808080  
Yellow RGB code = #FFFF00
