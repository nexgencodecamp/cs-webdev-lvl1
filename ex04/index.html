HTML tags

HTML tags are the building blocks of HTML. They are used to define the HTML elements that should appear on a web page.

Following is a list of some of the popular HTML 5 tags

<header> A <header> element will typically contain a heading for the section. It element could also contain navigation, search, logos, and other content.
<footer> A <footer> Footers typically contain information such as the author of the document, copyright information, links to terms of use, privacy policy, etc.
<nav> A <nav> is primarily intended for sections that consist of major navigation blocks.
<p> A paragraph text inserted between the start and end tags.
<ul> A <ul> tag is used to list items as an unordered format.
<ol> An <ol> tag is used to list items as an ordered list.
<div> The <div> element accepts content, which refers to most elements that can appear inside a document's body. It should only be used as a last resort when there isn't another suitable HTML element to use. HTML5 has introduced a number of new elements that can be used in place of the <div>. These include <article>, <aside>, <header>, and <footer>
<section> Any given web page or article could have many sections. For example, a homepage could have a section for introducing the company, another section for news items, and another section for contact information.
<article> An <article> is a self-contained, composition in a webpage that is reusable
<a> An <a> tag can be used to link text or images.
<img> An <img> tag is used for embedding images into an HTML document.
<table> An <table> tag is used for defining a table in an HTML document. The <table> tag contains other tags that define the structure of the table.
